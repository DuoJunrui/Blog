

# 使用Spring Boot开发一款简约的个人博客
> 作者 : 多俊睿

### 个人博客功能：

![](https://raw.githubusercontent.com/DuoJunrui/Blog/master/doc/个人博客功能脑图.jpg)

### 技术组合：

*  后端：Spring Boot + JPA + Thymeleaf模板
*  数据库：MySQL
*  前端UI：Semantic UI框架

### 工具与环境：

*  IDEA
*  Maven 3
*  JDK 8
*  Axure RP 8
